== Text Icons (default)

NOTE: A simple note with default text icon.

TIP: A simple tip with default text icon.

WARNING: A simple warning with default text icon.

CAUTION: A simple caution with default text icon.

IMPORTANT: An important message with default text icon.

== Text Icons with Custom Captions

:note-caption: Nota Bene
:tip-caption: Pro Tip
:warning-caption: Watch Out
:caution-caption: Careful!
:important-caption: Critical

NOTE: Note with custom caption from document attribute.

TIP: Tip with custom caption from document attribute.

WARNING: Warning with custom caption.

CAUTION: Caution with custom caption.

IMPORTANT: Important with custom caption.

== Block-level Caption Override

[caption="Read This"]
NOTE: Block-level caption overrides note-caption document attribute.

[caption="Insider Secret"]
TIP: Block-level caption overrides tip-caption document attribute.
